# Single-Page HTML CV

A clean, semantic, and well-structured single-page Curriculum Vitae (CV) built using **only HTML**. This project focuses strictly on document structure, SEO best practices, and social sharing optimization before adding any styling.

This project is part of the [roadmap.sh](https://roadmap.sh/projects/single-page-cv) frontend developer projects.

---

## 🚀 Features

- **Semantic HTML5:** Uses proper structural tags (`<header>`, `<main>`, `<section>`, `<article>`, `<footer>`) to ensure high accessibility and readability.
- **SEO Optimized:** Includes essential `<meta>` tags like description, keywords, and author in the `<head>`.
- **Open Graph (OG) Tags:** Fully configured for structured and visually appealing previews when shared on social media platforms (LinkedIn, Twitter, Facebook).
- **Favicon Integration:** Includes a custom favicon for a professional browser tab appearance.
- **Clean Structure:** Organized sections dedicated to:
  - Contact Information / Header
  - Professional Summary
  - Work Experience / Career History
  - Education
  - Skills
